# Chicken Hunt

## Description
Chicken Hunt — це 2D гра, в якій гравець стріляє по птахах та набирає очки.

## Features
- 2 типи птахів:
  - Звичайні (1 постріл)
  - З бронею (потрібно 2 постріли)
- 6 spawn point'ів для появи птахів
- Система очок за знищення
- Сцена з різними шарами (background, foreground)

## Mechanics
- Використовується Raycast для визначення попадання
- Птахи рухаються з випадковою швидкістю
- Броня (сковорідка) поглинає перший постріл

## Core Loop
1. Птахи спавняться
2. Гравець стріляє
3. Отримує очки
4. З’являються нові птахи
